# Twicome ユースケース

## アプリ概要

Twicome は Twitch VOD に寄せられたコメントをデータベース化し、検索・分析・可視化するウェブアプリケーションです。AI・機械学習を活用した高度な検索機能と、コミュニティエンゲージメント機能を提供します。

---

## ユーザーペルソナ

| ペルソナ | 説明 | 主な目的 |
|---|---|---|
| **配信者** | Twitch でライブ配信を行う人 | ファンの反応・コメント傾向を深く分析したい |
| **視聴者・ファン** | 特定の配信者や配信参加者を追う人 | 推しユーザーのコメント履歴や個性を探索したい |
| **コミュニティ管理者** | 配信コミュニティを運営する人 | 不適切なコメントの把握、コミュニティの健全性モニタリング |
| **研究者・分析者** | データ分析・自然言語処理を行う人 | Twitch コメントのセマンティック分析・感情変化の調査 |

---

## ユースケース一覧

### UC-01: コメント検索・閲覧

**概要**: 特定の Twitch ユーザーのコメント一覧を表示し、キーワードや条件でフィルタリングする。

**操作フロー**:
1. トップページでユーザー名を入力して検索
2. `/u/{login}` でそのユーザーのコメント一覧を表示
3. キーワード・配信者・VOD・日付でフィルタリング
4. 無限スクロールでコメントを閲覧

**利用シーン**:
- 過去の配信で話題になったコメントを遡る
- 特定のユーザーがいつ・どの配信で何を言ったかを確認する

---

### UC-02: セマンティック検索

**概要**: キーワードに意味的に類似したコメントをベクトル検索で抽出する。

**操作フロー**:
1. ユーザーコメントページで検索モードを「セマンティック」に切り替え
2. 検索ワード（例: 「ネタバレ」「感動した」）を入力
3. 入力したワードの意味に近いコメント群が表示される

**利用シーン**:
- 「ゲームのストーリーに感動したコメント」を全件検索せずに抽出したい
- 完全一致しない言い回しでも同じ意図のコメントを見つけたい

---

### UC-03: 感情軸検索

**概要**: 喜び・信頼・驚き・恐れ・悲しみ・嫌悪・怒りなど感情次元でコメントを絞り込む。

**操作フロー**:
1. 検索モードを「感情」に切り替え
2. 感情スライダー（例: 喜び 80%、驚き 40%）を調整
3. その感情プロファイルに合致するコメントが表示される

**利用シーン**:
- 視聴者が「興奮・驚き」を示したコメントだけを抽出したい
- ネガティブな感情（怒り・嫌悪）が高いコメントを確認したい

---

### UC-04: 典型度検索

**概要**: そのユーザーの「典型的なコメント」または「個性的なコメント」を抽出する。

**操作フロー**:
1. 検索モードを「典型度」に切り替え
2. スライダーで「典型側」「個性側」を調整
3. ユーザーのコメント重心からの距離に基づいて結果が表示される

**利用シーン**:
- そのユーザーがよく言うフレーズ・口癖を発見したい
- 普段と違う珍しいコメントをピックアップしたい

---

### UC-05: コメントクラスタリング

**概要**: ユーザーのコメントをテーマ別に自動分類し、どんな話題について多く発言しているかを視覚的に把握する。

**操作フロー**:
1. `/u/{login}/clusters` を開く
2. クラスタマップが表示され、テーマごとにコメントがグループ化される
3. クラスタをクリックしてサブクラスタ・コメント詳細を確認

**利用シーン**:
- ファンがどの話題（ゲーム・雑談・配信技術など）に多く言及しているか分析したい
- コミュニティの関心トピックを俯瞰したい

---

### UC-06: ユーザー統計分析

**概要**: ユーザーのコメント活動を時間帯・曜日・配信者別に可視化する。

**操作フロー**:
1. `/u/{login}/stats` を開く
2. コメント頻度の時間帯分布・曜日分布グラフを確認
3. どの配信者への言及が多いかを把握

**利用シーン**:
- ファンの活動パターン（何時ごろ配信を見ているか）を把握したい
- 複数の配信者をまたいで参加しているユーザーの行動を分析したい

---

### UC-07: コミュニティノート閲覧

**概要**: AI（OpenRouter API）が生成した補足情報・ファクトチェック・危険度スコアをコメントに表示する。

**操作フロー**:
1. コメント一覧でノートアイコンが付いているコメントを確認
2. ノートを開くと AI が生成した補足情報・危険度スコアが表示される

**利用シーン**:
- 不正確な情報やネタバレを含むコメントを素早く識別したい
- コミュニティ管理者が問題のある発言を確認したい

---

### UC-08: コメントへの投票

**概要**: コメントにいいね / よくないねを付けてコミュニティの評価を集積する。

**操作フロー**:
1. コメント一覧でいいね・よくないねボタンをクリック
2. 集計された票数が即時反映される

**利用シーン**:
- 優れたコメント・面白い発言をコミュニティで評価したい
- Best 9 展示に向けて高評価コメントを選出したい

---

### UC-09: Best 9 ショーケース

**概要**: 高評価を獲得したコメントをカード形式で一覧表示する。

**操作フロー**:
1. `/best9` を開く
2. 投票数上位のコメントが並んで表示される

**利用シーン**:
- コミュニティの名言・ハイライトコメントを振り返りたい
- 配信の盛り上がりポイントを記録・共有したい

---

### UC-10: クイズ（ユーザー識別ゲーム）

**概要**: 表示されたコメントがどのユーザーの発言か当てるランキングタスクゲームを楽しむ。

**操作フロー**:
1. `/u/{login}/quiz` を開く
2. コメントが提示され、候補ユーザーをランク付けして回答
3. 正解率・スコアが表示される

**利用シーン**:
- ユーザーの文体・個性を知り尽くしているかを試したい
- コミュニティメンバー同士の交流・遊びとして活用したい

---

### UC-11: コメントエクスポート

**概要**: ユーザーのコメントデータを CSV / JSON 形式でダウンロードする。

**操作フロー**:
1. ユーザーコメントページのエクスポートボタンをクリック
2. 形式（CSV / JSON）を選択してダウンロード

**利用シーン**:
- 外部ツールで独自分析を行いたい研究者・分析者
- コメントデータをアーカイブ・保存したい配信者

---

### UC-12: ユーザー追加・監視対象管理

**概要**: 新たな Twitch ユーザーを監視対象として登録し、コメント収集を開始する。

**操作フロー**:
1. `/add_user` でユーザーの Twitch ログイン名を入力
2. バッチ処理が VOD 一覧を取得してコメントをデータベースへ格納
3. FAISS インデックスが構築されセマンティック検索が利用可能になる

**利用シーン**:
- 新しく応援し始めた配信者の VOD コメントを遡って分析したい
- コミュニティ管理者が監視対象ユーザーを追加したい

## データフロー概要

```
Twitch API
    │
    ▼
VOD一覧取得 (get_vod_list_batch.py)
    │
    ▼
コメントDL (TwitchDownloader CLI)
    │
    ▼
DBインポート (insertdb.py) ──→ MySQL
    │
    ▼
埋め込みベクトル生成 (SentenceTransformer)
    │
    ▼
FAISSインデックス構築 ──→ FAISS マイクロサービス
    │
    ▼
Twicome Web アプリ (FastAPI)
    │
    ├── キーワード検索 (MySQL LIKE / fulltext)
    ├── セマンティック検索 (FAISS)
    ├── 感情軸検索 (埋め込みベクトル空間)
    ├── クラスタリング (クラスタメタデータ)
    └── コミュニティノート (OpenRouter API)
```

| URL | 説明 |
|---|---|
| `/` | トップページ・ユーザー検索 |
| `/u/{login}` | ユーザーコメント一覧・検索 |
| `/u/{login}/stats` | ユーザー統計分析 |
| `/u/{login}/clusters` | コメントクラスタリング |
| `/u/{login}/quiz` | クイズゲーム |
| `/vods` | VOD 一覧 |
| `/vods/{vod_id}` | VOD コメント一覧 |
| `/best9` | 高評価コメントショーケース |
| `/streamers` | 登録配信者一覧 |
| `/add_user` | ユーザー追加 |
| `/manual` | 使い方ガイド |
